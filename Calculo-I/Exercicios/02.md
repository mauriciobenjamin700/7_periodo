1) Resolva as inequações.

a) 3x + 3 < x + 6

**Resposta:**

- Começamos subtraindo **x** de ambos os lados da igualdade

(3x - 3x + 3) < (x - 3x + 6)

(3 < -2x + 6)

- Agora subtraimos 3 de ambos lados

(3 - 3) < (-2x + 6 - 3)

0 < (-2x + 3)

- Agora somamos +2x para igualar

(0 + 2x) < (-2x + 2x + 3)

2x < 3

- Agora dividimos por 2 para eliminar o 2x

(2x / 2) < (3 / 2)

- **Resultado:** **x < 1.5**

c) 2x − 1 ≥ 5x + 3

**Resposta:**

- (2x - 5x - 1) >= 5x -5x + 3

- (-3x -1) >= 3

- (-3x -1 + 1) >= (3 + 1)

- -3x >= 4

- -x >= 4/3

- -x >= 1,33

- **Resultado:** **x >= -1.33**

## Teorema do Confronto

Sejam f, g, h três fun~ções e suponha que existe r > 0 tal que `f(x) <= g(x) <= h(x)`

```math
para todo x E R, com 0 < |x - xo| < r.
```

com isso , se:

```math
lim(f(x), x->xo) = lim(j(x), x->xo) = L
```

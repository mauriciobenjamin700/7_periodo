## (ii) xo != 0
Dado **Epsilon > 0**, precisamos encontrar **Delta > 0** tal que:
    **|x - xo| < Delta -> |f(x) = xo| < Epsilon**

- a == 2
- b == 4

- 4 - 16 == 2 * 6

# Limite
Dada uma função **f**, dizemos que:   
- **lim f(x) = L**
- **x->xo**
quando:
- Vcortado Epsilon > 0, Einvertido Delta > 0 tal que:
- **0 < |x-xo| < Delta -> |f(x)-L| < Epsilon**

## Exemplo:
- **lim(3x - 1)**.
- **x->1**
- logo lim(1) ~= 2
